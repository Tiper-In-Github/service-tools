## 介绍

- 按需引入的分布式微服务和web服务工具包

- 这是一个实验性质的，用于分布式微服务和web服务的常用工具集合，主要提供模块化组件
- 当前是不稳定的，请勿使用

## 特点
- 模块化
- 零外部依赖

## 环境
- 推荐在go:1.20+的环境中使用

## 安装
```bash
go get github.com/GohperBridge/service-tools
```
